# Archived docs

Historical and AI-generated documentation produced during development.
Kept for reference — **not** required for setup or deployment.

For setup and deploy instructions, see [`../../SETUP.md`](../../SETUP.md)
at the repo root.

Broad categories of what lives here:

- `ACCESSIBILITY_*` — a11y audit reports and test summaries
- `AUTH*_TESTS_*`, `*_E2E_TESTS*`, `INTEGRATION_TESTS_*`, `TEST_SUMMARY.md`,
  `QA_*`, `PLAYWRIGHT_SETUP.md`, `E2E_SETUP_COMPLETED.md` — test
  documentation and verification artifacts
- `COLOR*`, `COLORS_AND_TYPOGRAPHY.md`, `DESIGN_*`, `LAYOUT_SPACING_*`,
  `SPACING_QUICK_REFERENCE.md`, `MICRO_INTERACTIONS_*`, `MOBILE_RESPONSIVE_REPORT.md`
  — design-system references and validation reports
- `CREATURE_*`, `INDEX_CREATURES.md`, `REDESIGN_*` — creature system docs
  and art redesign history
- `DATABASE_*` — database infrastructure notes (the *active* quickstart is
  `QUICK_DATABASE_SETUP.md` at the repo root)
- `DEPLOYMENT_VERIFICATION*`, `FINAL_DEPLOYMENT_AUDIT.md`,
  `PRODUCTION_APPROVAL_SUMMARY.md` — deployment verification artifacts
  (the *active* deploy docs are `DEPLOYMENT_GUIDE.md` and
  `DEPLOYMENT_CHECKLIST.md` at the repo root)
- `*_IMPLEMENTATION.md`, `IMPLEMENTATION_SUMMARY.md`,
  `PHASE_2_COMPLETION_SUMMARY.txt`, `PROJECT_COMPLETION_SUMMARY.txt`,
  `SUSPENSION_IMPLEMENTATION.md`, `UPDATES_IMPLEMENTATION.md`,
  `NOTIFICATIONS_PREFERENCES_IMPLEMENTATION.md`,
  `REPLY_NOTIFICATIONS_IMPLEMENTATION.md` — feature implementation notes
- `SECURITY_*` — security review and hardening reports
- `PERFORMANCE_*` — performance measurements
- `ISLA_COMPOSER_SETUP.md`, `QUICK_REFERENCE.md` — misc

Feel free to delete anything here that's no longer useful.
